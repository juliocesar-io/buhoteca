Sobre el proyecto

Proyecto final para el curso de base de datos, le proyecto es completamente Open Source , la idea se puede proyectar para impletarlo en la universidad ya que esta no cuenta con un sistema de catalogo y prestamo eficiente para los recursos de la biblioteca.

Si se animan a contribuir pueden leer la documentación para que entiendan la estructura del proyecto (como funcionan los modelos, la base de datos diseño y conexión, las vistas, etc..), el repositorio esta disponible en Github.
